What To Learn Before The Next Run

Pretrain Gap

Pretrain learned SVG syntax and primitive drawing patterns, but not the full visual vocabulary of the shipped assets. Your real targets are text-heavy, grouped, gradient-rich, and path-driven. Primitive-only pretrain leaves the model without that grammar.

Midtrain Gap

Midtrain introduced charts, cards, tables, and gradients, which is why control improved early. But it still did not teach explicit numeric data payloads, multi-panel composition, or reconstruction tasks from structured specs.

SFT Gap

SFT data taught shallow tag control more than intentful composition. `pretrain_1` was the best balance point. Later runs narrowed or distorted the control contract, so syntax stayed stable while instruction following regressed. `sft_7` recovers some of that, but not enough yet.

Next Clean Run

For the next chain, use a stable spec-to-SVG prompt contract, seed from the strongest practical checkpoint, and add asset-derived pretrain data plus render-level canary evals. That is the shortest path toward production-quality infographics.
